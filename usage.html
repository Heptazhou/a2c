YAAW 使用说明

JSON-RPC Path
  • JSON-RPC Path 默认为：http://localhost:6800/jsonrpc
  • 如果提示 Aria2 RPC 服务器错误 按照以下方法修改
    • host: 指运行 Aria2 的设备所在的 IP 或者名字
    • port: 使用 --rpc-listen-port 选项设置的端口，默认为 6800
    • 一般设置为：http://host:port/jsonrpc
    • 使用 --rpc-secret=xxxxxx 选项时设置为：http://token:xxxxxx@host:port/jsonrpc
    • 其中 http 可以用 ws 替代，使用 WebSocket 协议：ws://token:xxxxxx@host:port/jsonrpc
Tips
  • 在 YAAW 中对 Aria2 的设置会在 Aria2 重启后丢失，必要的设置请写入配置文件（默认路径为：~/.config/aria2/aria2.conf）
  • 下载完成的任务会在 Aria2 重启后消失，除非启用了 --force-save 选项
  • 因界面已汉化，其他不再赘述
